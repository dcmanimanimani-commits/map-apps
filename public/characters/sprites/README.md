# アバター歩行スプライト（9キャラ × 16枚）

アバターたんけん用の歩行コマ画像をここに置きます。

## 枚数

- **9キャラ**（avatar-1 〜 avatar-9）
- **各16枚** = 4方向 × 4コマ

## フォルダ構成

```
sprites/
  avatar-1/
    up-idle.webp
    up-walk1.webp
    up-walk2.webp
    up-walk3.webp
    right-idle.webp
    …（計16枚）
  avatar-2/
  …
  avatar-9/
```

## 画像の作り方

- **背景は透過**（PNG または透過 WebP）。白地のまま export しないこと。
- 白背景の画像を置いた場合は `npm run sprites:transparent` で自動変換できる（既存10枚も適用済み）。
- キャラの後ろは透明にして、地図の上に自然に重ねる。
- 推奨: 正方形キャンバス、キャラは中央寄せ、余白は透明。
- ファイル名は上記どおり（方向-コマ名）。拡張子 `.webp` 推奨。

## コマの意味

| ファイル | 用途 |
|---------|------|
| `*-idle.webp` | 止まっている |
| `*-walk1.webp` | 歩き1（右足前など） |
| `*-walk2.webp` | 歩き2（左足前など） |
| `*-walk3.webp` | 歩き3（予備・将来用） |

### avatar-5（ケガしたハムスター）の注意

- **包帯はキャラの左足（画面では向かって右）に固定**。`idle` と同じ位置のまま。
- `walk1`（右足前）でも `walk2`（左足前）でも、包帯は**前に出した足ではなく左足**に巻く。
- 前足に包帯が付いたり、コマで包帯の足が入れ替わらないようにする。

方向: `up` `right` `down` `left`（画面上向き基準）
